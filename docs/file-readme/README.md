# InternTrack File-by-File Explanations

This folder contains simple explanations for each main source file.

How to use this:
1. Open the same path under src.
2. Open its matching .md file here.
3. Read Purpose first, then Syntax Notes, then Flow.

Naming rule:
- Source file: src/pages/LoginPage.jsx
- Explanation file: docs/file-readme/src/pages/LoginPage.jsx.md

These notes are written for beginner-level React + Firebase understanding.
